# House Buying Calculator

Live: https://evim-hesapla.vercel.app/

Minimal calculator for Turkish housing loan rules based on BDDK. The application is entirely built with vibe coding. 

## Stack
- React + TypeScript + Vite
- Tailwind CSS
- Vitest + Testing Library
- GitHub Actions (build & test)
